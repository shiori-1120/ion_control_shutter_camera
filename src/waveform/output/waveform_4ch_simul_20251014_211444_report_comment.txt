# ==================================================================
# 【測定基準の定義 (Measurement Criteria)】
#
# このレポートの各数値は、以下の基準に基づいて計算されています。
#
#  - V_high / V_low:
#    ノイズやオーバーシュートの影響を避けるため、電圧データの
#    上位5%と下位5%を除外した範囲での最大値・最小値です。
#
#  - Period (周期) / Jitter (ジッター):
#    信号の立ち上がりエッジが「振幅の50%」の電圧レベルを
#    通過する時刻を基準点とし、その基準点同士の時間間隔を測定しています。
#    Periodはその平均値、Jitterはそのばらつきです。
#
#  - Rise Time (立ち上がり時間):
#    信号の立ち上がりエッジが「振幅の10%」から「振幅の90%」に
#    到達するまでの時間を測定しています。
#
#  - Fall Time (立ち下がり時間):
#    信号の立ち下がりエッジが「振幅の90%」から「振幅の10%」に
#    到達するまでの時間を測定しています。
#
#  - Skew (スキュー):
#    基準チャンネル(CH1)の立ち上がりエッジが「振幅の50%」を通過した
#    時刻を基準(0秒)とし、他のチャンネルの立ち上がりエッジ(50%点)が
#    それに対してどれだけ時間的にズレているかを測定しています。
#
# ==================================================================
